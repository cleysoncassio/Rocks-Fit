---------------------------------------------------------
   ROCKS-FIT: MÓDULO DE RECEPÇÃO PROFISSIONAL
---------------------------------------------------------

Este módulo gerencia o acesso via reconhecimento facial e 
controle direto da catraca.

ARQUIVOS PRINCIPAIS:
1. MODO_LIMPO.vbs      <- USE ESTE para iniciar o monitor (Modo Profissional)
2. INICIAR_ROCKSFIT.bat <- Inicia o monitor com janela de log (Para manutenção)
3. ENCERRAR_SISTEMA.bat <- Use para fechar o sistema completamente
4. ponte_rocksfit.py    <- O código principal do sistema

COMO INSTALAR (NOVO PC):
1. Instale o Python 3.12+ (Marque "Add to PATH")
2. Na pasta do projeto, crie o ambiente virtual:
   python -m venv .venv
3. Instale as dependências:
   .venv\Scripts\pip install -r MODULO_RECEPCAO\requirements_recepcao.txt
4. Para usar, basta clicar em MODO_LIMPO.vbs

DICA:
Crie um atalho do 'MODO_LIMPO.vbs' na Area de Trabalho 
para facilitar o acesso diário.
