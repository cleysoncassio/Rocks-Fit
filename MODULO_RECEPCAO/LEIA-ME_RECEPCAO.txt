🚀 MODULO DE RECEPÇÃO ROCKS FIT (PONTE LOCAL)
===========================================

Este módulo foi organizado para ser executado no computador da recepção da academia.

ARQUIVOS INCLUÍDOS:
------------------
1. ponte_rocksfit.py      -> Interface Principal (Monitor e Gestor).
2. leitor_bio.py          -> Ponte para o Leitor de Digital (U.are.U 4500).
3. teste_biometria.py     -> NOVO! Script dedicado para testar o hardware digital.
4. ponte_catraca.py       -> Interface de rede com o hardware da catraca.
5. diagnostico_rksfit.py  -> Ferramenta de diagnóstico completo (Caminhos, Redes, Bio).
6. rkslogo.png            -> Logo para a interface.
7. abrir_catraca.bat      -> Atalho para liberação manual de emergência.
8. ENCERRAR_SISTEMA.bat   -> Fecha todos os processos da Rocks Fit.
9. requirements_recepcao.txt -> Lista de dependências.

COMO INSTALAR:
-------------
1. Instale o Python 3.10+ no computador.
2. Com o leitor Digital Persona conectado, instale o "DigitalPersona OneTouch SDK" (Driver oficial).
3. No terminal (CMD) dentro desta pasta, rode:
   pip install -r requirements_recepcao.txt

COMO EXECUTAR (SISTEMA COMPLETO):
-------------------------------
Para o funcionamento total (Monitor + Digital), você deve abrir DOIS terminais:

Terminal A (Interface):
   python ponte_rocksfit.py

Terminal B (Leitor de Digital):
   python leitor_bio.py

CONFIGURAÇÃO DE IP:
-----------------
Se o IP da sua catraca mudar, edite o arquivo 'ponte_rocksfit.py' e ajuste:
CATRACA_IP = "169.254.37.150"

RESOLUÇÃO DE PROBLEMAS:
---------------------
- Leitor não acende: Verifique se o Driver DigitalPersona está instalado.
- Catraca não abre: Use o 'diagnostico_rksfit.py' para testar a comunicação de rede.
- Erro no PIP: Rode o CMD como Administrador.
